# Get Your Free Cohere API Key

## Quick Steps

1. Go to: https://dashboard.cohere.com/
2. Click "Sign Up" (free account)
3. Verify your email
4. Go to "API Keys" section
5. Copy your API key

## Add to `.env`

```bash
GOOGLE_API_KEY=your_existing_gemini_key
PINECONE_API_KEY=your_existing_pinecone_key
INDEX_NAME=your_existing_index_name
COHERE_API_KEY=your_new_cohere_key_here
```

## Why Cohere?

- ✅ **Generous free tier**: 100 API calls/minute
- ✅ **Fast embeddings**: `embed-english-light-v3.0` model
- ✅ **384 dimensions**: Smaller, faster than OpenAI
- ✅ **No credit card required**

Once you add the key, we can run the system!
